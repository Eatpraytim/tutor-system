Standard Normal (Z) Table — Cumulative Probability P(Z ≤ z)

Read: row = z to first decimal, column = second decimal. Example: z = 1.96 → row 1.9, col .06 → 0.9750

Student's t-Distribution — Critical Values (Two-Tailed)

Reject H⊂0 if |t| > critical value. For one-tailed, halve α.

df = n − 1 for one-sample test, df = n − 2 for simple regression, df = n − k − 1 for multiple regression.

Chi-Square (χ²) Distribution — Critical Values (Right Tail)

Reject H⊂0 if χ² > critical value. df = (rows−1)(cols−1) for independence, df = k−1 for goodness of fit.

F-Distribution — Critical Values (α = 0.05)

Reject H⊂0 if F > critical value. df1 = numerator (k−1 for ANOVA), df2 = denominator (n−k).
